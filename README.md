# Brand-Aware Ad Copy Generator

## Overview
This project explores how Large Language Models can be used to generate
structured ad copy for D2C brands while maintaining brand voice and
competitive differentiation.

## Problem Statement
Marketing teams often struggle to produce consistent, high-quality ad copy
that aligns with brand tone and differentiates from competitors.

## Solution
- Accepts brand tone, product details, and competitor context
- Uses structured prompting to generate multiple ad variations
- Allows quick experimentation through a simple UI

## Key Concepts Used
- Prompt engineering
- LLM creativity evaluation
- Structured output generation

## Tech Stack
- Python
- LangChain
- OpenAI
- Streamlit

## Notes
Built as a self-initiated experiment to understand LLM-driven creative workflows
in performance marketing.
